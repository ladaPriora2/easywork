1. nona -- для автоматического заполнения пропусков с помощью моделей машинного обучения(работает только с числовыми атрибутами);
2. XGBoost -- для моделей градиентного бустинга ("градиентный бустинг сына маминой подруги": умеет паралеллиться на несколько ядер, оптимизированный, может заменить все остальные модели в 85% случаев);
3. seaborn -- для высокоуровневых и красивых визуализаций (если лень или не умеете в matplotlib);
4. geopandas -- для работы с геоданными (по сути, тот же pandas (даже операции pandas можно применять  к GeoDataFrame), но может считывать файлы с расширениями *.geojson, *.shp, *.dbf, *.shx);
5. openCV и scikit-image -- для работы с изображениями (как раз на тот самый страшный день, если данные будут содержать набор изображений);
6. dtreeviz -- ещё немного визуализаций, но для деревьев принятия решений (выглядит интереснее, чем стандартная plot_tree);
7. pretty-confusion-matrix -- для отображения красивой матрицы неточностей (не придётся писать функцию для её визуализации самому);
8. pypdf -- для считывания pdf;
9. num2words -- для преобразования чисел в слова (для задач NLP);
10. top2vec -- для визуализации самых частых слов в тексте;
11. datefinder -- на тот случай, если понадобится извлечь из текста даты и время;
12. pmdarima -- библиотека с моделью машинного обучения для временных рядов;
13. sqlalchemy -- библиотека для работы с БД на SQL.
